# Images Folder

## ⚠️ Action Required

Add your profile photo here!

## Profile Photo
- **Filename:** `profile.jpg` (must match exactly)
- **Recommended size:** 400×400px minimum (square ratio)
- **Formats supported:** JPG, PNG, WebP
- **Tips:** Use a clear headshot; the image is displayed in a circular frame

If no image is found, a placeholder with instructions is shown instead.

## To use a different filename or format:
Open `index.html` and find:
```html
<img src="images/profile.jpg" alt="Vaishnav N" class="profile-img" ...>
```
Change `profile.jpg` to your filename (e.g., `profile.png`, `vaishnav.jpg`).

## Adding Other Images (e.g., blog thumbnails)
Store all images here and reference them in `index.html` as:
```html
<img src="images/your-image.jpg" alt="description">
```
